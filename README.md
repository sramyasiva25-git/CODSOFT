# CODSOFT Python Internship Projects

This repository contains multiple Python-based projects completed as part of my Python Programming Internship.

Each task focuses on improving core Python skills such as logic building, functions, loops, and user interaction.

##  Tasks Overview

| Task No. | Project Title | Description | Key Concepts | Technologies |
|---------|---------------|-------------|--------------|--------------|
| Task 1 | To-Do List Application | Add, view, and delete tasks using a menu-driven program | Lists, Functions, Loops | Python |
| Task 2 | Simple Calculator | Performs basic arithmetic operations based on user choice | If-Else, Functions | Python |
| Task 3 | Password Generator | Generates random passwords using letters, numbers, and symbols | Random Module, Strings | Python |
| Task 4 | Rock Paper Scissors Game | User vs computer game with score tracking | Loops, Game Logic | Python |
| Task 5 | Contact Book Application | Manages contacts with add, search, update, and delete options | Lists, Dictionaries | Python |
