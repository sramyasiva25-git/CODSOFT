# Contact Book Application
##  Description

This project is a Contact Book application developed using Python.
It allows users to store, view, search, update, and delete contact details using a menu-driven console interface.

##  Features

  - Add new contacts

  - View all saved contacts

  - Search contacts by name or phone number

  - Update existing contact details

  - Delete contacts

  - Simple and user-friendly menu

##  Technologies Used

  - Python

##  How to Run

  - Open the project folder in VS Code

  - Run the following command in the terminal:

  - python contact_book.py


  - Choose an option from the menu and follow the instructions
