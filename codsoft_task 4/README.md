# Rock Paper Scissors Game
## Description

   This project is a Rock Paper Scissors game developed using Python.
The user plays against the computer, and the program determines the winner based on the game rules while maintaining a scoreboard.

## Features

  - User vs Computer gameplay

  - Random choice generated by computer

  - Displays winner for each round

  - Maintains user and computer scores

  - Option to play again or exit

## Technologies Used

  - Python

  - Open the project folder in VS Code

  - Run the following command in the terminal:

  - Enter your choice (rock / paper / scissors)

  - View result and score after each round
